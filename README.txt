Приложение для работы со студентами.
Цель - приложение со всеми элементами структуры (репозиторий, сервисный слой, API) для отработки навыков тестирования.

Студенты имеют следующие связи:
- Город
- Сданные предметы с оценкой
Классический CRUD по всем сущностям.

Ряд имплементаций добавлен для дальнейшего развития проекта.
Работать можно через swagger или postman.
http://localhost:8080/webapp/swagger-ui/index.html?configUrl=/webapp/v3/api-docs/swagger-config#/city

Работа с БД осуществляется при помощи flyway (ddl генерация отключена).
Структура БД без доп оптимизация типа индексов и т.п. (ключи - простые числа для упрощения понимания)
Cвязь с БД через соответствующий профиль конфигурации в application.yml. В этом же файле прочие настройки
